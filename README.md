# 🔌 Arduino Ammeter

Simple Arduino-based ammeter using a shunt resistor.

## ⚙️ Components

* Arduino Uno
* Shunt Resistor
* A0, A1 Pins

## ⚡ Working

Measures voltage drop across shunt and calculates current:

I = V / R

## 🚀 Usage

1. Upload code
2. Connect circuit
3. Open Serial Monitor
